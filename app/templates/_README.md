# Generator Html5 Site
A generator for Html5 Site.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator **locally**: `npm install generator-html5-site`
- Run: `yo html5-site`
