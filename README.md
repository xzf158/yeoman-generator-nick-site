# Generator Html5 Site
A simple generator for Yeoman.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator **locally**: `npm install -g generator-nick-site`
- Run: `yo nick-site`
